De PollTool

De "Poll tool" was een school project wat we voor een opdrachtgever binnen de school moesten maken. Hierin zat onder andere een inlog systeem, een random code generator en met deze code kon je inloggen als leerling dan. De back-end hiervan heb ik dan ook vooral gemaakt, dus de code generator, het inlog systeem etc. Het moeilijke hiervan vond ik het werken met die speciale codes maar voor de rest vond ik het niet erg moeilijk om te maken.
