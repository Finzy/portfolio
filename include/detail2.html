The poll tool

De "Poll tool" was een school project wat we voor een opdrachtgever binnen de school moesten maken. Het onderwerp van het project was om de ideale "tool" te maken voor docenten over de hele wereld. Wij hadden een tool gemaakt waar docenten in kunnen loggen om hier een poll te maken waar ze een vraag konden invullen, als de docent dit had gedaan kreeg hij een code die hij aan zijn klas kan geven zodat zij de code kunnen invoeren op de "leerlingen" sectie op de website. Nadat ze deze hebben ingevuld krijgen zij de vraag voor zich en kunnen hier antwoord op geven. De docent krijgt dan op zijn eigen profiel de antwoorden van zijn vraag.
