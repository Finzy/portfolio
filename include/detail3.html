Over de schutting

Dit was een school project wat we moesten maken in een groepje van 2. We moesten een website na maken die 2 jaar geleden was gemaakt. Alle content van de website stond in een photoshop bestand en we moesten het daar allemaal uit halen om de website zo identiek mogelijk te maken. Hiervoor moest ook een hoop documentatie geschreven worden. This was a school project that we had to make in groups of 2. We had to duplicate a website that was up 2 years ago. We got all the content in a photoshop file and had to use that to make the website the same as possible. There was alot of documentation involved aswell.
